# VETO - Диктатор Решений

Мобильное приложение для группового принятия решений с механикой "рулетки" и системой вето.

## Описание

VETO помогает группам друзей быстро принимать решения (куда пойти, что делать) без споров. Приложение использует механику рулетки для случайного выбора и систему вето для отмены нежелательных решений.

## Основные функции

✅ **Авторизация**
- Быстрый старт (анонимный вход)
- Регистрация с email и паролем
- Вход с email и паролем

✅ **Группы**
- Создание групп
- Вход в группу по коду
- Просмотр участников и их токенов вето

✅ **Рулетка решений**
- Добавление вариантов выбора
- Удаление своих вариантов
- Запуск рулетки с анимацией
- Случайный выбор варианта

✅ **Система вето**
- 3 токена вето на пользователя
- Отмена решения рулетки
- Realtime уведомления всех участников

## Технологический стек

- **Frontend**: Flutter 3.x
- **State Management**: Riverpod 2.x (с кодогенерацией)
- **Backend**: Supabase
  - PostgreSQL база данных
  - Supabase Auth для авторизации
  - Realtime для WebSocket уведомлений
- **Роутинг**: go_router
- **UI**: Brutal Design System

## Настройка проекта

### 1. Установка зависимостей

```bash
flutter pub get
```

### 2. Настройка Supabase

1. Создайте проект на [supabase.com](https://supabase.com)
2. Перейдите в SQL Editor и выполните скрипт из `supabase_schema.sql`
3. Скопируйте URL и Anon Key из Settings → API
4. Вставьте их в `lib/core/config/supabase_config.dart`:

```dart
static const String supabaseUrl = 'YOUR_SUPABASE_URL';
static const String supabaseAnonKey = 'YOUR_SUPABASE_ANON_KEY';
```

### 3. Генерация кода Riverpod

```bash
flutter pub run build_runner build --delete-conflicting-outputs
```

### 4. Запуск приложения

```bash
flutter run
```

- `options` - Варианты выбора

- Realtime синхронизация для всех изменений

## Использование

### 1. Регистрация/Вход
- Быстрый старт: введите имя
- Или: зарегистрируйтесь с email

### 2. Создание группы
- Нажмите "СОЗДАТЬ ГРУППУ"
- Введите название
- Поделитесь кодом с друзьями

### 3. Запуск рулетки
- Добавьте минимум 2 варианта
- Нажмите "ЗАПУСТИТЬ РУЛЕТКУ"
- Дождитесь результата

### 4. Использование вето
- Нажмите "ИСПОЛЬЗОВАТЬ ВЕТО"
- Подтвердите действие
- Токен будет списан, рулетка запустится заново
